# Be Faster Than a Speeding Ticket

A 3D Unity game where you talk your way out of traffic tickets using AI-powered conversations.

[![Gameplay Video](https://img.youtube.com/vi/guKcdQ-0OOY/0.jpg)](https://youtu.be/guKcdQ-0OOY)

[Play Demo on itch.io](https://dream-decypher.itch.io/be-faster-than-a-speeding-ticket-supercell-global-ai-game-hack?secret=SFuMAAY4Mk6oK9Bf6SEZEoU3H0)

## Features

- **AI-Powered NPC Dialogue** — Real-time conversations with AI cops using Gemini, ChatGPT, or Mistral as backend providers
- **Voice Input (Speech-to-Text)** — Talk to NPCs using your microphone, with WebGL browser support
- **Text-to-Speech** — NPCs speak back with unique voices per character via Deepgram TTS
- **AI-Generated 3D Assets** — Characters and props created with Nimble Fox AI 3D generation
- **Multiple Characters & NPCs** — Each character has a distinct personality type that affects NPC behavior and dialogue
- **Dynamic Leniency System** — A real-time progress bar tracks how likely the cop is to let you go, with color-coded feedback (red/yellow/green)
- **Economy System** — Start with cash, lose money per ticket, try to survive as long as possible
- **Sequential Gameplay** — Each character meets every NPC in order before rotating to the next
- **Cinematic Transitions** — Cutscene intros with smooth scene transitions and camera work
- **WebGL Ready** — Fully playable in the browser with custom microphone plugin

## Tech Stack

- Unity 3D (URP)
- AI Providers: Google Gemini / OpenAI ChatGPT / Mistral AI
- TTS: Deepgram Aura
- STT: Mistral Voxtral
- 3D Assets: Nimble Fox
- Platform: WebGL, Desktop
